# Create: Sophisticated Storage Compatibility

This mod provides compatibility between the Create mod and Sophisticated Storage mod for Minecraft 1.20.1 on Fabric.

## Features

- Enables automatic item transfers between Create's mechanical systems (belts, depositors) and Sophisticated Storage's advanced chests.
- Checks for the presence of both mods and enables compatibility only when both are loaded.

## Setup

For setup instructions, please see the [Fabric Documentation page](https://docs.fabricmc.net/develop/getting-started/creating-a-project#setting-up) related to the IDE that you are using.

## Building

Run `./gradlew build` to build the mod.
